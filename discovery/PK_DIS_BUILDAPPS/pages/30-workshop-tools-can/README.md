# What workshop-tools can do

You can use workshop-tools to help you run an event:
- Distribute credentials and an link to the participants via mail
- Notify to the CSP of their customer participation
- Mark the attendance of participants
- Together with _workshop-tools_, you can use Docsify to make the exercises available to the participants. With it you can:
    - Use placeholders for the participant's number and credentials
    - Use placeholders to add forms within the exercise to collect the participant results
- Verify exercise completion and track participant progress
- Set unique passwords for the participant credentials (for IAS users)
- Send participation data to _repository-tools_
