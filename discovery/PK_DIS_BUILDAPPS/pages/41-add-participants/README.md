# Adding participants to a workshop

## Using the Upload Participants app

You can use the tool to upload participants via excel (Recommended).

- Make sure your excel has a table on the first sheet, and that the table starts in A1
- Make sure your excel contains the columns (names can be with or without space, and lower or upper case, order does not matter, and values can be blank):
  - First name
  - Last name
  - Email
  - CSP name
  - CSP email
  - Approved/Registration Status. _True values are: Approved, true, yes, 1, y, x. Any other will be false_
  - CRM id
  - Customer name
  - Comments (optional)

Your excel can have additional columns, those will be ignored
