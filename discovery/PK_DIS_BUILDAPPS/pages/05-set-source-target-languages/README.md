23. Click on **Input Field 1**.

 * Navigate to the **Properties** tab.
 * Enable the **Multiline Input** option by setting it to **True**.

   * Click on **Input Field 2**
   * Go to the **Properties** tab
   * Set **Multiline Input** to **True**


![](im/69.png)

24. Enter the text you want to translate into another language in the input field, and then click **Translate** button.

> **Example text:**
> SAP Translation Hub is an AI-powered service on SAP Business Technology Platform (BTP) that provides fast and high-quality translation for SAP software, custom applications, and documents. It leverages SAP’s multilingual repository, machine translation capabilities, and company-specific data to enable seamless localization and global collaboration. The service supports translation of UI texts, documentation, and other content using SAP’s expertise, machine learning models, and custom glossaries. It integrates easily into business workflows through secure APIs, ensuring efficient and reliable translation.

![](im/70.png)

You can observe that the text has been successfully translated into the target language.

25. Select Source and Target Languages

You can select any supported language as the **source** or **target** language (highlighted in the interface). Once the languages are selected, click the **Translate** button.
The text will be translated automatically into the chosen target language.

![](im/71.png)
